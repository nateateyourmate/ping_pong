# ping_pong
Testing a ping pong game run on PyGame's framework.

Playable local multiplayer game that is playable with two players. (Or yourself if you don't have friends)
- For the left player, move up and down using the W and S keys.
- For the right player, move up and down using the corresponding up and down arrow keys.

To win, deflect the ball to the opposing player's side. If they messed up and couldn't block it in time, then you win.
